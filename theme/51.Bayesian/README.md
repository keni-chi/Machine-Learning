# Bayesian

ベイズ最適化 (Bayesian Optimization, BO) 
    Gaussian Process Regression (GPR)を利用する。
    ガウス過程による回帰により説明変数と目的変数との間で回帰モデルを構築する
    推定値とその分散を利用して、目的変数の値がより大きくなる(小さくなる)可能性の高いサンプル候補を見つける
    実験計画法やハイパーパラメータの最適化に活用できる
