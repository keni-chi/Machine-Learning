# クラスタリング

- 階層的クラスタリング  

- k平均法 (k-means clustering) 非階層的クラスタリング  

- 混合ガウスモデル（GMM)
    概要
        与えられたデータセットを、複数の正規分布の重ね合わせで表現する
        確率密度関数が得られる (確率分布として表現できる)
        サンプルごとに、各クラスターに所属する確率が得られる
        クラスター数を自動的に決められる
    利用方法
        クラスターの数を自動で決めたい
        データセットの確率密度が必要な時
    詳細
        クラスター数を振って、ベイズ情報量規準（BIC)の値が最小となるクラスタ数とする。
